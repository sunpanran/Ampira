# Ampira 新标签页

短描述：从书签里的网站获取新资讯，按时间或重要性查看。AI 功能可选，可生成摘要和今日简报，也能结合正文继续解读。数据默认保存在本地，不用注册账号。

收藏夹里有不少以后想看的网页，真正再打开的并不多。关注的网站也散在不同地方，很难每天逐个查看。Ampira 读取你选择的书签，把保存的网页和这些网站的新内容放进新标签页，并按时间或重要性整理。需要时，AI 会生成卡片摘要和今日简报，也能读取正文并继续回答问题。

如果你需要长期跟踪一个行业或研究主题，可以把常看的来源放进同一个书签文件夹。Ampira 会集中更新，按重要性排序，再用 AI 整理成今日简报。遇到值得继续看的内容，也可以打开正文追问。

## 主要功能

- Ampira 读取你选择的 Chrome 书签文件夹，把书签作为主要资讯来源；也可以补充 RSS、Atom、JSON Feed 和公共来源（Ampira）。书签始终只读，Ampira 不会创建、修改或删除原书签。
- 资讯可以按时间或重要性排序。同一事件的相关内容会放在一起；需要回看时，可以按日期浏览或在归档中继续查找。
- 配置自己的 API Key 后，AI 可以生成卡片摘要和今日简报。粘贴网址或针对已收录内容提问后，也能结合正文继续回答。AI 功能可选，不配置时资讯获取、排序、稍后读和阅读器照常使用。
- 每天从灵感预设（Ampira）或你选择的书签文件夹显示一组灵感卡片，也可以随时“换一换”。
- 点击工具栏图标，或从资讯和书签卡片把网页加入稍后读。站内阅读只显示提取后的正文、图片和视频链接，不运行远程脚本或嵌入播放器；搜索和归档可以找回已收录内容。
- 快捷书签、待办、天气、顶部浏览器搜索和深浅色主题也可按需使用。

## 数据与权限

- 不用注册账号。Ampira 没有广告、行为分析或开发者服务器。
- Feed、阅读状态和缓存保存在浏览器中。稍后读、待办和天气城市默认只保存在本地，需要时可以分别开启 Chrome Sync。
- AI 与图片搜索 API Key 只保存在当前 Chrome 配置文件中，不会同步，也不会发送给开发者。使用对应功能时，密钥只会随请求直接发送给你选择的服务商。
- 网站访问权限只会在你启用来源或需要读取网页时申请，并限制到明确的来源域名。权限可以在设置中撤销。

安装后 Ampira 会替换 Chrome 新标签页。要恢复默认页面，请在扩展管理页停用或卸载 Ampira。
