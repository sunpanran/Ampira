# Ampira 新标签页

短描述：把选择的 Chrome 书签与资讯来源整理成可搜索的新标签页信息看板。

## 详细描述

Ampira 把散落在 Chrome 书签里的资讯源、灵感站点和常用入口整理成一个高密度新标签页：今日信号、事件主题、阅读队列、资讯流、归档索引和可选 AI 摘要一屏完成。

- 本地优先：没有账号、广告、分析或开发者服务器。
- 一键稍后读：点击 Ampira 工具栏图标，把当前网页保存到本地阅读队列，并在浮窗中确认结果。
- 逐来源授权：只在用户操作后申请具体网站域名。
- 公共 Feed 补盲：内置公开来源默认开启，可在设置中关闭，并且只读取用户明确授权的具体域名。
- 安全阅读：远程网页提取为结构化正文、图片和视频入口，不执行远程脚本或嵌入播放器。
- 本机密钥：AI 与图片搜索密钥只保存在当前 Chrome 配置的扩展本地存储中，不参与同步。
- 三种语言、深浅色主题与响应式布局。

安装后 Ampira 会替换 Chrome 新标签页。要恢复默认页面，请在扩展管理页停用或卸载 Ampira。
