# Ampira 新分頁

簡短說明：把選取的 Chrome 書籤與資訊來源整理成可搜尋的新分頁資訊看板。

## 詳細說明

Ampira 將 Chrome 書籤中的資訊源、靈感網站與常用入口整理成高密度新分頁：今日訊號、事件主題、閱讀佇列、資訊流、封存索引與可選 AI 摘要集中完成。

- 本機優先，沒有帳號、廣告、分析或開發者伺服器。
- 一鍵稍後讀：點擊 Ampira 工具列圖示，把目前網頁儲存到本機閱讀佇列，並在浮窗中確認結果。
- 只在使用者操作後按網域申請來源權限。
- 公共 Feed 補充功能預設開啟，可在設定中關閉，並且只讀取使用者明確授權的特定網域。
- 安全閱讀會提取結構化正文、圖片與影片入口，不執行遠端程式碼或嵌入播放器。
- AI 與圖片搜尋金鑰只保存在目前 Chrome 設定檔的擴充功能本機儲存空間，不參與同步。
- 支援英文、簡體中文與繁體中文介面，以及深色、淺色主題。

安裝後 Ampira 會取代 Chrome 新分頁。要恢復預設頁面，請在擴充功能管理頁停用或解除安裝 Ampira。
