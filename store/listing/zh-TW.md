# Ampira 新分頁

簡短說明：從書籤裡的網站取得新資訊，依時間或重要性查看。AI 功能可選，可產生摘要與今日簡報，也能讀取內文並繼續解讀。資料預設留在本機，不用註冊帳號。

收藏夾裡存了不少打算以後再看的網頁，真正回頭開啟的卻不多。關注的網站也散在不同地方，很難每天逐一查看。Ampira 讀取你選擇的書籤，把保存的網頁和這些網站的新內容放進新分頁，再依時間或重要性整理。需要時，AI 會產生卡片摘要與今日簡報，也能讀取內文並繼續回答問題。

如果你需要長期追蹤一個產業或研究主題，可以把常看的來源放進同一個書籤資料夾。Ampira 會集中更新，依重要性排序，再用 AI 整理成今日簡報。遇到值得繼續看的內容，也可以開啟內文追問。

## 主要功能

- Ampira 讀取你選擇的 Chrome 書籤資料夾，把書籤作為主要資訊來源；也可以補充 RSS、Atom、JSON Feed 與公共來源（Ampira）。書籤始終維持唯讀，Ampira 不會建立、修改或刪除原有書籤。
- 資訊可以依時間或重要性排序。同一事件的相關內容會放在一起；需要回看時，可以依日期瀏覽或在封存中繼續搜尋。
- 設定自己的 API Key 後，AI 可以產生卡片摘要與今日簡報。貼上網址或針對已收錄內容提問後，也能結合內文繼續回答。AI 功能可選，不設定時資訊取得、排序、稍後讀與閱讀器照常使用。
- 每天從靈感預設（Ampira）或你選擇的書籤資料夾顯示一組靈感卡片，也可以隨時「換一換」。
- 點擊工具列圖示，或從資訊與書籤卡片把網頁加入稍後讀。站內閱讀只顯示擷取後的內文、圖片與影片連結，不會執行遠端程式碼或嵌入播放器；搜尋與封存可以找回已收錄內容。
- 快捷書籤、待辦、天氣、頂部瀏覽器搜尋和深淺色主題也可依需要使用。

## 資料與權限

- 不用註冊帳號。Ampira 沒有廣告、行為分析或開發者伺服器。
- Feed、閱讀狀態與快取保存在瀏覽器中。稍後讀、待辦和天氣城市預設只保存在本機，需要時可以分別開啟 Chrome Sync。
- AI 與圖片搜尋 API Key 只保存在目前的 Chrome 設定檔中，不會同步，也不會傳送給開發者。使用對應功能時，金鑰只會隨請求直接傳送給你選擇的服務商。
- 網站存取權限只會在你啟用來源或需要讀取網頁時申請，並限制在明確的來源網域。權限可以在設定中撤銷。

安裝後 Ampira 會取代 Chrome 新分頁。要恢復預設頁面，請在擴充功能管理頁停用或解除安裝 Ampira。
