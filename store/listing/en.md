# Ampira New Tab

Short description: Turn Inspiration preset (Ampira), selected Chrome bookmarks, and feeds into a searchable new-tab dashboard.

## Detailed description

Ampira combines the ready-to-use Inspiration preset (Ampira) with bookmarked news sources, personal inspiration sites, and everyday links in a dense new-tab workspace with daily signals, topic clusters, a reading queue, feed cards, an archive index, and optional AI summaries.

- Local-first: no account, ads, analytics, or developer-operated backend.
- Ready-made inspiration: choose 48 curated public sites with 24 packaged original covers, or switch to your existing inspiration bookmark folder at any time. Ampira never writes bookmarks.
- Read later: add news and bookmark cards shown in Ampira to a local queue, with optional Chrome Sync.
- Per-origin access: website permissions are requested only after a user action.
- Public-Feed supplement: built-in public sources are enabled by default, can be turned off in Settings, and are fetched only for exact origins the user grants.
- Safe reader: remote pages are extracted into structured text, images, and video links without running remote scripts or embedded players.
- Local credentials: optional AI and image-search keys stay in extension-local storage for the current Chrome profile and are never synchronized.
- Optional content sync: reading queue, to-dos, and selected weather city stay local by default and can each be enabled separately for Chrome Sync.
- English, Simplified Chinese, and Traditional Chinese UI with light and dark themes.

Ampira replaces Chrome’s New Tab page while the extension is enabled. Disable or uninstall it from Chrome’s extension manager to restore the default page.
