Support

Restore Chrome’s default New Tab page

Open chrome://extensions, find Ampira, then disable or uninstall it. Chrome does not allow an extension page setting to remove its own New Tab override dynamically.

A news source has no content

Check source coverage and per-source diagnostics on the News page, then verify exact-domain access on the Browser page. If a source page declares a Feed on another domain, Ampira lists that exact domain and waits for your click before requesting it. Bookmark entries remain available without access.

AI is unavailable

Confirm that an API key is saved, the data-transfer notice is accepted, the AI service domain is allowed, and the connection test succeeds. Ampira does not provide or proxy API keys.

Report a problem

Use the project support channel and include reproduction steps, your Chrome version, and screenshots that contain no keys or private bookmarks.
