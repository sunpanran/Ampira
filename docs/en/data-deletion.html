Data deletion

  1. On the News page in Settings, select Clear cache to remove news, summaries, search caches, and source-quality records.
  2. On the AI page in Settings, select Remove key or Remove image key to delete the corresponding credential from extension-local storage.
  3. Uninstall Ampira from chrome://extensions to remove its IndexedDB, API keys, and local extension storage.
  4. If Chrome Sync still retains settings or opted-in reading queue, to-dos, or weather city, first turn off the corresponding category under Settings → Browser → Cross-device content, then remove any remaining extension data through Chrome’s synchronized-data controls.

Ampira has no developer-operated account or server, so there is no cloud user database for the developer to delete.
