Data deletion

  1. Under Settings → About → Import & export, select Factory reset to delete Ampira settings, API keys, consent, the local cover, caches, and user content on this device; remove optional permissions; and delete Ampira settings plus opted-in reading queue, to-dos, and weather city from browser account sync. Synced deletions affect other devices on the same account but do not remotely erase local-only data unique to those devices. Browser bookmarks are not changed.
  2. On the News page in Settings, select Clear cache to remove news, summaries, search caches, and source-quality records.
  3. On the AI page in Settings, select Remove key or Remove image key to delete the corresponding credential from extension-local storage.
  4. Uninstall Ampira from chrome://extensions in Chrome or edge://extensions in Microsoft Edge to remove its IndexedDB, API keys, and local extension storage.
  5. To keep other Ampira data, turn off an individual category under Settings → Browser → Cross-device content; this removes only that category’s browser account sync copy and preserves its local content.

Ampira has no developer-operated account or server, so there is no cloud user database for the developer to delete.
