支援

恢復預設新分頁

開啟 chrome://extensions，找到 Ampira 後停用或解除安裝。Chrome 不允許擴充功能透過頁面內設定動態取消新分頁覆寫。

資訊來源沒有內容

在 Ampira 設定的「瀏覽器」頁檢查對應來源網域是否已授權。未授權時書籤項目仍可使用，但擴充功能不會讀取 Feed 或正文。

AI 無法使用

確認已儲存 API 金鑰、已同意資料傳輸說明、已授權 AI 服務網域，並在設定中測試連線。Ampira 不提供或代理 API 金鑰。

問題回報

請透過專案支援入口提交重現步驟、Chrome 版本，以及不含金鑰或私人書籤的螢幕截圖。
