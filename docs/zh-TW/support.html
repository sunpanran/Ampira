支援

恢復預設新分頁

在 Chrome 開啟 chrome://extensions，或在 Microsoft Edge 開啟 edge://extensions，找到 Ampira 後停用或解除安裝。瀏覽器不允許擴充功能透過頁面內設定動態取消新分頁覆寫。

資訊來源沒有內容

先在「資訊」頁查看來源覆蓋率與逐源診斷，再到「瀏覽器」頁檢查對應來源網域是否已授權。若來源頁宣告了另一個網域上的 Feed，Ampira 會列出該精確網域並等待你點擊授權；授權前不會請求它。未授權時書籤項目仍可使用。

AI 無法使用

確認已儲存 API 金鑰、已同意資料傳輸說明、已授權 AI 服務網域，並在設定中測試連線。Ampira 不提供或代理 API 金鑰。

問題回報

請透過專案支援入口提交重現步驟、瀏覽器名稱與版本，以及不含金鑰或私人書籤的螢幕截圖。
