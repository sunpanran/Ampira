資料刪除

  1. 在「設定 → 關於 → 設定移轉」點選「恢復原廠設定」，可刪除目前裝置上的 Ampira 設定、API 金鑰、授權、本機首圖、快取和使用者內容，撤銷選用權限，並刪除瀏覽器帳戶同步中的 Ampira 設定及使用者選擇同步的閱讀佇列、待辦和天氣城市。同步刪除會影響同一帳號的其他裝置，但不會遠端刪除其他裝置獨有的本機資料；瀏覽器書籤不會被修改。
  2. 在設定的「資訊」頁點選「清除快取」，刪除資訊、摘要、搜尋快取與來源品質記錄。
  3. 在設定的「AI」頁點選「移除金鑰」或「移除搜尋金鑰」，刪除保存在擴充功能本機儲存空間中的對應憑據。
  4. 在 Chrome 的 chrome://extensions 或 Microsoft Edge 的 edge://extensions 解除安裝 Ampira，會清除擴充功能 IndexedDB、API 金鑰與本機擴充功能儲存空間。
  5. 若不希望恢復全部資料，可先在「設定 →瀏覽器 → 跨裝置內容」關閉單項同步開關，只刪除對應內容的瀏覽器帳戶同步副本並保留本機內容。

Ampira 沒有開發者帳號或伺服器，因此不存在需要向開發者申請刪除的雲端使用者資料庫。
