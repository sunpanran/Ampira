資料刪除

  1. 在設定的「資訊」頁點選「清除快取」，刪除資訊、摘要、搜尋快取與來源品質記錄。
  2. 在設定的「AI」頁點選「移除金鑰」或「移除搜尋金鑰」，刪除保存在擴充功能本機儲存空間中的對應憑據。
  3. 在 chrome://extensions 解除安裝 Ampira，會清除擴充功能 IndexedDB、API 金鑰與本機擴充功能儲存空間。
  4. 若 Chrome Sync 仍保留設定或使用者選擇同步的閱讀佇列、待辦、天氣城市，可先在「設定 → 瀏覽器 → 跨裝置內容」關閉對應內容開關，再透過 Chrome 的同步資料管理清除剩餘擴充功能資料。

Ampira 沒有開發者帳號或伺服器，因此不存在需要向開發者申請刪除的雲端使用者資料庫。
