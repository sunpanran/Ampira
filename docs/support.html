支持

恢复默认新标签页

在 Chrome 打开 chrome://extensions，或在 Microsoft Edge 打开 edge://extensions，找到 Ampira 后停用或卸载。浏览器不允许扩展通过页面内设置动态取消新标签页覆盖。

资讯来源没有内容

先在“资讯”页查看来源覆盖率与逐源诊断，再到“浏览器”页检查对应来源域名是否已授权。若来源页声明了另一个域名上的 Feed，Ampira 会显示该精确域名并等待你点击授权；授权前不会请求它。未授权时书签仍可使用。

AI 无法使用

确认已保存 API 密钥、已同意数据传输说明、已授权 AI 服务域名，并在设置中测试连接。Ampira 不提供或代理 API 密钥。

问题反馈

请通过项目支持入口提交复现步骤、浏览器名称与版本，以及不含密钥或私人书签的截图。
