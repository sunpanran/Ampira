支持

恢复默认新标签页

打开 chrome://extensions，找到 Ampira 后停用或卸载。Chrome 不允许扩展通过页面内设置动态取消新标签页覆盖。

资讯来源没有内容

在 Ampira 设置的“浏览器”页检查对应来源域名是否已授权。未授权时书签入口仍可用，但扩展不会读取 Feed 或正文。

AI 无法使用

确认已保存 API 密钥、已同意数据传输说明、已授权 AI 服务域名，并在设置中测试连接。Ampira 不提供或代理 API 密钥。

问题反馈

请通过项目支持入口提交复现步骤、Chrome 版本和不含密钥或私人书签的截图。
