数据删除

  1. 在“设置 → 关于 → 配置迁移”点击“恢复出厂设置”，可删除当前设备上的 Ampira 设置、API 密钥、授权、本地头图、缓存和用户内容，撤销可选权限，并删除浏览器账户同步中的 Ampira 设置及用户选择同步的阅读队列、待办和天气城市。同步删除会影响同一账号的其他设备，但不会远程删除其他设备独有的本机数据；浏览器书签不会被修改。
  2. 在设置的“资讯”页点击“清除缓存”，删除资讯、摘要、搜索缓存和来源质量记录。
  3. 在设置的“AI”页点击“移除密钥”或“移除搜图密钥”，删除保存在扩展本地存储中的对应凭据。
  4. 在 Chrome 的 chrome://extensions 或 Microsoft Edge 的 edge://extensions 卸载 Ampira，会清除扩展 IndexedDB、API 密钥和本机扩展存储。
  5. 若不希望恢复全部数据，可先在“设置 →浏览器 → 跨设备内容”关闭单项同步开关，仅删除对应内容的浏览器账户同步副本并保留本机内容。

Ampira 没有开发者账号或服务器，因此不存在需要向开发者申请删除的云端用户数据库。
