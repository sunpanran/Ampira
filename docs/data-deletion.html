数据删除

  1. 在设置的“资讯”页点击“清除缓存”，删除资讯、摘要、搜索缓存和来源质量记录。
  2. 在设置的“AI”页点击“移除密钥”或“移除搜图密钥”，删除保存在扩展本地存储中的对应凭据。
  3. 在 chrome://extensions 卸载 Ampira，会清除扩展 IndexedDB、API 密钥和本机扩展存储。
  4. 若 Chrome Sync 仍保留设置或用户选择同步的阅读队列、待办、天气城市，可先在“设置 → 浏览器 → 跨设备内容”关闭相应内容开关，再通过 Chrome 的同步数据管理清除剩余扩展数据。

Ampira 没有开发者账号或服务器，因此不存在需要向开发者申请删除的云端用户数据库。
